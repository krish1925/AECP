Integrations

Drop-in adapters for popular vector databases and frameworks.

ChromaDB Stable

Full adapter: serve-mode EmbeddingFunction + offline migration with batched I/O and recall sampling.

pip install "aecp[chroma]"
from aecp.adapters.chroma import AECPChromaFunction

ef = AECPChromaFunction(mapping, new_model_embedder=embed_fn)
col = client.get_collection("docs", embedding_function=ef)
results = col.query(query_texts=["search"], n_results=10)

LangChain Stable

Drop-in Embeddings shim. Works with any LangChain vector store.

pip install "aecp[langchain]"
from aecp.adapters.langchain import AECPEmbeddings

ae = AECPEmbeddings(mapping, base_embeddings)
db = Chroma.from_documents(docs, embedding=ae)
results = db.similarity_search("query", k=10)

Qdrant Beta

Stream-read adapter for Qdrant collections.

pip install "aecp[qdrant]"
from aecp.stores.qdrant_store import QdrantStore

store = QdrantStore.connect(url="http://localhost:6333")
for batch in store.iter_vectors(batch_size=1024):
    Z = m.transform(np.stack([r.vector for r in batch]))

Sentence Transformers Stable

Direct integration for local model inference.

pip install "aecp[sentence-transformers]"
from sentence_transformers import SentenceTransformer

old = SentenceTransformer("all-MiniLM-L6-v2")
new = SentenceTransformer("BAAI/bge-large-en-v1.5")

X = old.encode(calibration_texts)
Y = new.encode(calibration_texts)
m = RidgeMapping(alpha="auto", seed=0).fit(X, Y)
